User Documentation
==================

Our user documentation provides users of grommunio with the resources to work
and use grommunio. With emphasis on easy-to-read and up-to-date content, we try
to help users with all kinds of different skill levels to interact with
grommunio.

+-------------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------------------+
| `grommunio User Docs <https://docs.grommunio.com/user>`_                | Using grommunio with different software clients, such as Microsoft Outlook, Mozilla Thunderbird and more.                           |
+-------------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------------------+
| `grommunio Web Docs <https://docs.grommunio.com/web>`_                  | Using grommunio Web, including references to grommunio Meet, grommunio Files and grommunio Chat.                                    |
+-------------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------------------+

Administrator Documentation
===========================

Our administrator documentation helps administrators to install, configure and
operate grommunio and its components. For administrators of grommunio, it is
recommended to have at least a very basic skillset with operating with the
Linux operating systems, though it is not mandatory to understand all the
details.

+-------------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------------------+
| `grommunio Administrator Docs <https://docs.grommunio.com/admin>`_      | Main grommunio Administrator Documentation covering all aspects of grommunio, from installation to operation.                       |
+-------------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------------------+
| `grommunio Man Pages <https://docs.grommunio.com/man>`_                 | Main grommunio Man Pages collection for Administrators, with details for each service and configurable parameter.                   |
+-------------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------------------+
| `grommunio Migration Docs <https://docs.grommunio.com/migration>`_      | This document provides information on migration strategies to grommunio, mainly for use by Administrators.                          |
+-------------------------------------------------------------------------+-------------------------------------------------------------------------------------------------------------------------------------+
